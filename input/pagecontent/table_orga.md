Le champ **ORC-17 – Entering Organization** est requis par le profil IHE RAD – Scheduled Workflow (SWF) afin de qualifier l’organisation à l’origine de l'ordre véhiculer en ORC.  

Dans le cadre du volet Téléradiologie, ce champ est utilisé pour **indiquer le type d’organisation** émettrice, sans pour autant véhiculer l’identifiant complet de celle-ci.  
L’identification structurée et pérenne de l’organisation est portée dans **ORC-21 – Ordering Facility Name**, de type **XON**, conformément aux principes du CI-SIS.
